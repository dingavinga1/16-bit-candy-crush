# 16-bit-candy-crush
A multi-level implementation of "Candy Crush" in 16-bit Assembly Language (MASM)!

## Sneak Peek
![Level_2](https://user-images.githubusercontent.com/88616338/182585444-9fee909d-cba3-43b1-a6ad-971e6a1eba4b.png)


## Features
- #### Main menu
- #### 3 levels with different difficulty levels
- #### Swapping with mouse
- #### Bomb integration!

## How to Try
- Download MASM6.15 from [here](http://www2.hawaii.edu/~pager/312/masm%20615.ZIP)
- Copy the contents to ```C:\```
- Download DOSBOX from [here](https://sourceforge.net/projects/dosbox/files/dosbox/0.74-3/DOSBox0.74-3-win32-installer.exe/download)
- Open DOSBOX Options and add the following lines at the end
  - ```mount c c:\masm615\bin```
  - ```c:```
- Copy the candy.asm file from this repository to ```C:\masm615\bin```
- Run DOSBOX and enter the following commands to start playing
  - ```ml candy.asm```
  - ```candy.exe```
